# Landing Page Profil - Kelompok 3

Proyek ini adalah tugas mata kuliah Rancang Bangun Perangkat Lunak untuk membuat sebuah website Landing Page Profil Kelompok dengan mengimplementasikan sistem Input-Proses-Output (IPO) sederhana.

## Anggota Kelompok & Pembagian Tugas
* **Nandika Aditia** - Koordinasi tim & Struktur Halaman utama (index.html)
* **Rafly Affansyah** - Styling, UI Design & Layout (style.css)
* **Zaki Baharuna** - Styling, UI Design & Layout (style.css)
* **M. Nur Robbany** - Animasi & Interaksi Logika IPO (script.js)
* **Muhammad Rofik Aprizani** - Dokumentasi Project & Skenario Pengujian (README.md & tests/test.md)

## Teknologi yang Digunakan
* HTML5
* CSS3
* Vanilla JavaScript

## Cara Menjalankan Proyek
1. Clone repositori ini ke komputer lokal Anda.
2. Buka folder proyek di text editor (contoh: Visual Studio Code).
3. Jalankan file `index.html` menggunakan browser atau ekstensi Live Server.
